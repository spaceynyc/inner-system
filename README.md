# THE INNER SYSTEM

A high-end, interactive 3D landing page built with React Three Fiber.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Features
- **Frosted Glass Centerpiece**: Interactive icosahedron with `MeshTransmissionMaterial`.
- **Deep Space Atmosphere**: Custom fog and background.
- **Floating Typography**: 3D text integration.
- **Interactive UI**: Minimalist play button overlay.

## Tech Stack
- React
- Vite
- React Three Fiber
- Drei
- Framer Motion
- Maath
